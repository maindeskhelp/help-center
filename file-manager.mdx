## Using the File Manager

The File Manager serves as the central hub for all your photo and file management needs. It allows you to easily upload, organize, and access all of your files and images in one place.

## Accessing the File Manager

To access the File Manager, simply navigate to Website Settings -\> File Manager. From there, you can choose between two options: All Files or All Images.

If you only need to upload images, select All Images. This option automatically resizes and compresses your images, making the process similar to uploading via the Image Widget. However, if you need to upload any file type or an image in its original size and quality, choose All Files.

## Uploading Files and Images

Uploading your desired image or file is a breeze with the File Manager. Simply click the 'Upload a file' button and select the file you want to upload. The File Manager even allows you to create custom folders to keep your files and images organized.

After uploading, each file will generate a unique URL link. This link can be used to add a downloadable file to your eCommerce store, or for other purposes.

## Conclusion

The File Manager is a powerful tool for managing all of your files and images in one convenient location. Whether you need to upload images for your website or add downloadable files to your eCommerce store, the File Manager has got you covered. So next time you need to manage your files, remember to access the File Manager for a seamless experience.
