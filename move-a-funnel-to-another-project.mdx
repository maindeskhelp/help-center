# How to Move a Website or Funnel to a Different Project

When you move a funnel or website to a different project, it will share the same URL as the original item. This means that you no longer have to worry about managing separate custom domains for each project.

 

In addition, all contacts, orders, and automations associated with the funnel or website will also be shared between the projects. This is a great way to keep consistent branding and data across your projects.

 

If you have any further questions or need assistance with this process, please don't hesitate to reach out to our support team. We are always happy to help!
